# Corpo em Progresso

Aplicativo em Expo/React Native para uso privado de duas pessoas, com perfis separados, histórico local, fotos dentro do app, gráficos e backup em JSON.

## Instalação

```bash
npm install
npx expo start
```

## Recursos incluídos

- dois perfis: Você e Esposa
- peso, gordura, cintura, quadril, braço, coxa
- foto de evolução por registro
- cards no painel
- gráficos simples
- histórico com exclusão
- meta de peso e IMC
- exportação/importação de backup em JSON

## Observação importante

O backup em JSON exporta os dados do app. As fotos permanecem no armazenamento local do app. Para um backup completo com fotos, a próxima etapa é gerar um pacote ZIP.
